# Mouse Laboratory Management System — Prototype

ระบบบริหารจัดการข้อมูลสัตว์ทดลอง (หนู) — **Interactive Prototype v0.1**

โปรโตไทป์แบบคลิกได้จริงทุกหน้า สร้างด้วย HTML + CSS + JavaScript ล้วน **ยังไม่เชื่อมฐานข้อมูลจริง** (ข้อมูลเป็น mock ในหน่วยความจำ) เป้าหมายคือทดสอบ **UX/Workflow** กับผู้ใช้งานจริงในห้องแลปก่อน แล้วจึงค่อยพัฒนาระบบจริงและต่อฐานข้อมูลในลำดับถัดไป

> UI เป็นภาษาไทย · ข้อมูลรีเซ็ตทุกครั้งที่รีเฟรชหน้า (ตั้งใจให้เป็นเช่นนั้นในเฟสนี้)

## วิธีเปิดใช้งาน
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย — ไม่ต้องติดตั้งอะไร ไม่ต้องมีเซิร์ฟเวอร์หรือ build step

```
เปิด index.html → เลือกบทบาทเข้าสู่ระบบ → เริ่มใช้งาน
```

## ฟีเจอร์หลัก
- **เข้าสู่ระบบ + สลับบทบาท** ได้ทันทีจากมุมขวาบน (EC, PI, Stock, Vet, นักวิทย์) เพื่อทดสอบสิทธิ์
- **รายการโครงการ** และ **สร้างโครงการใหม่** — ตั้งข้อมูล กลุ่มทดลอง และจัดผังกรง/หนูในหน้าเดียว (wizard เลือกกลุ่ม + เพศ + จำนวนต่อกรง)
- **Dashboard** จำลองตำแหน่งกรงจริงเป็นชั้น (Shelf) แต่ละกรงแสดงน้ำหนักหนูทุกตัว การเปลี่ยนแปลง และปริมาณน้ำ/อาหารที่กิน พร้อมสีสถานะ
- **โหมดชั่งน้ำหนัก** (เฉพาะนักวิทย์) — workflow ทีละกรง กรอกน้ำ/อาหาร/น้ำหนักหนูทีละตัว ด้วยแป้นตัวเลขบนจอ ออกแบบให้เร็วที่สุด
- **จัดการกรง/หนู** — เพิ่ม/แก้ไข/ลบกรง ปรับจำนวนหนู และแก้ผังโครงการภายหลังได้
- **งานสัตวแพทย์** (เฉพาะ Vet) — เพิ่มการรักษา ปิดเคส และสั่ง Humane endpoint (ระบุสาเหตุ)
- **บันทึกการตาย / Stop** ต่อหนูรายตัว (ระบุลักษณะการตาย + การจัดการซาก)
- **รายงาน & กราฟ** — เส้นน้ำหนัก/น้ำ/อาหาร เทียบรายกลุ่ม/รายตัว + Export CSV
- **Audit Log** — บันทึกกิจกรรมทั้งระบบ เปิดดูได้ทุกคนเพื่อความโปร่งใส

## สถานะสี
- **กรง**: เทา = ปกติ · เหลือง = กำลังรักษา/ดูแล
- **หนู** (ตามการเปลี่ยนแปลงน้ำหนักต่อวัน): เขียว = ขึ้นปกติ · เหลือง = ขึ้นน้อยกว่าเกณฑ์ · แดง = ลด/ไม่เพิ่ม

## โครงสร้างไฟล์
```
index.html        โครงหน้า + โหลด CSS/JS
css/styles.css    สไตล์ทั้งหมด (CSS variables, ไม่มี preprocessor)
js/data.js        mock data + helper (Data)
js/app.js         ตัวควบคุมทั้งแอป (App): routing, render, dialogs, wizard
CLAUDE.md         บันทึกสถาปัตยกรรม/กติกาโดเมนสำหรับผู้พัฒนา
```

## เทคโนโลยี
Vanilla HTML/CSS/JavaScript — ไม่มี dependency, ไม่มี framework, ไม่มี build step
ฟอนต์ Sarabun จาก Google Fonts · กราฟเป็น SVG เขียนเอง · Export เป็น CSV ฝั่ง client

## ยังไม่ทำในเฟสนี้ (ตามขอบเขต prototype)
ฐานข้อมูลจริง/persistence, KPI dashboard, Notification, ระบบสิทธิ์แบบละเอียด (บังคับใช้จริงเฉพาะการชั่งน้ำหนักและงานสัตวแพทย์), Import/Export จริง, เวอร์ชันมือถือ
